Detailed Explanation about The Project

  • Ticket booking is a process of selecting and making reservations for the cinema in a particular theatre via online.
    • Front-end: Reactjs
    • Back-end: Nodejs
    • Database: MongoDB
  • This application contains, the Admin user and Client user.
    • Admin User
      • Able to create/modify/delete theatres.
      • The particular theatre contains a number of shows, and each show can only have 100 seats.
      • Only Admin user are allowed to add films to the portal.
      • Admin user can view the queries given by the clients.
    • Client User
      • Only Authenticated users are allowed to book shows.
      • The users are allowed to select any number of seats.
      • The Client user can request his queries to the admin user via the contact us page.
      • Client users are secured with a log-in system with JWTtoken and OTP is sent to registered mail when trying to reset the password.
