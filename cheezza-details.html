Project Overview

I have developed a comprehensive MERN-based pizza ordering website that addresses the need for convenient online pizza orders. The project utilizes a modern tech stack, including Node.js, Express.js, MongoDB, React, Redux, Nodemon, Mailer, Multer, and Stripe for payment integration.

Tech Stack

The project leverages a powerful tech stack, including Node.js for server-side scripting, Express.js for building the backend, MongoDB as the database, and React for the frontend. Redux manages the state, while Nodemon ensures a smooth development experience. Mailer and Multer handle email functionality and file uploads, respectively. Stripe integration simplifies secure payments.

  • Authentication and Authorization
    • Implemented JWT token-based user authentication for secure access.
    • Role-based authorization distinguishes between admin and user roles.

  • Cookie-Based Authentication
    • Enhanced security and user convenience with cookie-based JWT token storage for persistent sessions.

  • Payment Integration
    • Integrated Stripe for secure and confident user payments.

  • Admin Dashboard
    • Admin dashboard for managing products, orders, and users.
    • Role-based access control ensures exclusive access for admin users under the profile icon.

  • File Upload
    • Utilized Multer to simplify user profile picture uploads, enhancing personalization.

  • Password Recovery
    • Implemented a secure "Forgot Password" feature with Mailtrap integration.
    • Generates secure reset tokens for improved account security and recovery.

  • Database Seeding
    • Data seeding offering a comprehensive range of pizza options from the start.

  • Error Handling
    • Implemented error handling using async/await and try-catch blocks for a better user experience.

  • Search Functionality
    • Streamlined the ordering process with a user-friendly search feature.

  • User Profile
    • Personalized user experience by enabling profile picture uploads and providing access to order history.
